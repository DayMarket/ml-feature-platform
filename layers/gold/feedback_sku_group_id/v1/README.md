# Gold-фичи отзывов по SKU Group ID

Пайплайн строит дневный снимок базовых фичей отзывов и рейтинга на уровне `sku_group_id`.

Целевая таблица: `iceberg.gold.feature_platform_sku_group_feedback_base_stats`.

Основная логика:

- читает опубликованные отзывы из `foodback.public.feedback`;
- обогащает отзывы SKU-данными из `` `dwh-iceberg`.silver.sku ``;
- собирает всю историю опубликованных отзывов до начала расчетной даты Airflow `ds`, то есть состояние на вчерашний день;
- агрегирует отзывы на уровень `sku_group_id`;
- считает средний рейтинг, хорошие и плохие отзывы;
- считает распределение отзывов по оценкам 1-5;
- считает количество отзывов с непустым текстом;
- считает доли оценок, плохих отзывов и хороших отзывов от общего количества отзывов;
- пишет результат в Iceberg через `overwritePartitions()`.

Партиция результата соответствует Airflow `ds`.
